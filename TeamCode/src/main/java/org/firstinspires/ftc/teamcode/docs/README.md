# Iron Reign Coding Documentation

Our coding guidelines and documentation are maintained on our team website:

**[https://ironreignrobotics.org](https://ironreignrobotics.org)**

## For Team Members

If you need to edit the documentation:

1. Clone the private `irdocs` repository
2. Make your changes
3. Push to `irdocs`
4. The website will be updated via subtree pull
